# Exploring the Effect of Hidden Layers on Multilayer Perceptron Performance and Overfitting

## Overview

This project investigates how the number of hidden layers in a Multilayer Perceptron (MLP) affects model performance, training efficiency, and overfitting. It also explores the use of early stopping as a regularisation technique to improve generalisation.

The study compares shallow and deep neural networks using experimental results and visualisations.

---

## Objectives

* Analyse how increasing hidden layers impacts accuracy
* Evaluate training time and computational cost
* Identify overfitting in deeper models
* Demonstrate the effectiveness of early stopping

---

## Features

* MLP models with 1, 3, and 5 hidden layers
* Performance comparison (train vs test accuracy)
* Loss curve visualisation
* Training time analysis
* Early stopping implementation

---

## Dataset

The project uses the **sklearn digits dataset**, which consists of:

* 8×8 pixel images
* 10 classes (digits 0–9)
